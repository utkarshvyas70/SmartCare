# Smart Healthcare Appointment System

A web-based healthcare appointment scheduling platform built using Next.js.
The system allows patients to book appointments with doctors while managing scheduling conflicts efficiently.

## Features
- Doctor appointment booking
- Conflict detection
- Doctor availability management
- Responsive UI

## Tech Stack
- Next.js
- React
- Tailwind CSS

## Learning Outcomes
- Component-based architecture in Next.js
- State management for scheduling logic
- UI optimization using Tailwind CSS
